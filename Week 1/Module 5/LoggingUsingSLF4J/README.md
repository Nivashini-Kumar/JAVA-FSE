# LoggingUsingSLF4J

## Objective

Learn how to use SLF4J logging instead of System.out.println().

## Concepts Covered

- SLF4J API
- Logger creation
- Logging levels

## Logging Levels

INFO
- General application information.

DEBUG
- Detailed developer information.

WARN
- Warning messages.

ERROR
- Error messages.

## Run

mvn test
