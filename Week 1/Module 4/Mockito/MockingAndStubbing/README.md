# MockingAndStubbing

## Objective

Learn how to use Mockito to create mock objects and define their behavior using stubbing.

## Concepts Covered

### Mocking
Creating fake objects instead of using real dependencies during testing.

### Stubbing
Defining what a mock object should return when a method is called.

## Mockito Methods Used

- mock()
  - Creates a fake object.

- when()
  - Defines the behavior of a mock object.

- thenReturn()
  - Specifies the value returned by the mock.
